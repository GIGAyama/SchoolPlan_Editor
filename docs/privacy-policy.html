本ポリシーは、週案作成支援アプリ「週案エディタ」（以下「本アプリ」）における利用者情報の取り扱いを定めるものです。運用者は {{運用者名}}（連絡先: {{連絡先メール}}）です。

1. 基本方針：利用者データは利用者本人の Google Drive にのみ保存されます

本アプリは Google Apps Script 上で「アクセスしているユーザーとして」動作します。週案・単元マスタ・タスク・学級通信・行事予定PDF などの入力データは、利用者本人の Google アカウント（Google ドライブ／スプレッドシート）内にのみ保存されます。運用者は、これらのデータを運用者側のサーバーやデータベースに収集・保存・閲覧しません。

2. 取得する情報とアクセス範囲（OAuth スコープ）

本アプリは機能提供のために、Google の認可（OAuth）で以下の範囲にアクセスします。いずれも利用者本人の資産に対する操作であり、運用者への送信・保管は行いません。

  • userinfo.email: ログイン中の利用者を識別し、各自のデータベースを分離するため。
  • spreadsheets: 利用者本人のスプレッドシート（週案データベース）の読み書き。
  • drive.file: 本アプリが作成した、または利用者が選択したファイルのみを対象とする Drive 操作（PDF の取り込み・出力・行事予定PDFの保存）。利用者の Drive 全体にはアクセスしません。
  • script.send_mail: 利用者が有効化した場合のみ、タスクのリマインダーメールを利用者本人宛に送信するため。
  • classroom.courses.readonly / classroom.announcements: 利用者が Google Classroom 連携を使う場合のみ、連携クラスの一覧取得と、利用者自身のクラスへのお知らせ投稿を行うため。

3. 第三者サービスの利用

  • Google（Workspace / Drive / Sheets / Classroom）: データの保存・処理は Google の基盤上で行われます。Google のプライバシーポリシーが適用されます。
  • 生成AI（Gemini API 等）: PDF 解析やタスク抽出のため、利用者が設定した API キーを用いて、対象の PDF/テキストを AI サービスへ送信する場合があります。送信内容・API キーは利用者の設定に基づくもので、運用者は保持しません。AI 提供者の利用規約・プライバシーポリシーが適用されます。

4. Google ユーザーデータの取り扱い（Limited Use）

本アプリによる Google ユーザーデータの利用は、Google API サービスのユーザーデータに関するポリシー（Limited Use 要件を含む）に準拠します。取得したデータは、上記の機能提供の目的にのみ利用し、広告目的での利用や、無関係な第三者への譲渡は行いません。

5. データの削除・撤回

  • 入力データは利用者の Drive 上にあるため、利用者がスプレッドシートを削除すれば消去されます。
  • 本アプリへのアクセス許可は、利用者の Google アカウントのセキュリティ設定からいつでも取り消せます。
